# Setup — Referência de Comandos

## init

```bash
# Inicializar novo projeto com estrutura do Task Master
task-master init
```

## models --setup

```bash
# Configurar modelos de LLM (executar após init)
task-master models --setup
```

## parse-prd

```bash
# Parsear arquivo PRD e gerar tasks
task-master parse-prd <prd-file.txt>
```
